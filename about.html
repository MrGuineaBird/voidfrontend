about void

search the web,
your way.

VOID is a free, non-profit search engine built by the creator of ROK. It runs locally on a personal machine — so don't expect the server to be online 24/7, ykyk. We're actively scaling our index, and as soon as that Amazon 500GB external hard drive arrives, it plans to be a genuinely helpful search engine for everyone.

// origin

Built by the creator of ROK

VOID started as a side project — a search engine that actually respects you. No ads, no tracking, no corporate nonsense. Just results.

// infrastructure

Locally ran, honestly

The server runs on a personal PC. That means sometimes it's offline. We're not going to pretend otherwise — it's part of the charm.

// index

Actively growing

Our crawler is running. The index is expanding. Once that 500GB drive lands, expect a serious jump in coverage across the open web.

// mission

Non-profit, always

VOID will never serve ads or sell your data. It exists because the web deserves a search engine that isn't trying to monetize your attention.

// what we believe

The open web is worth indexing. Search should be free — not just in price, but in spirit. No filters, no agendas, no ads. Just the web, as it is.
